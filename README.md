# Wisata APP (Nuxt3 + Vuetify3 + Pinia)

## Stacks

```
- vue3
- nuxt3
- vuetify3
- pinia
- nuxt-viewport
```

## Project setup

```
# yarn
yarn

# npm
npm install

# pnpm
pnpm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev

# pnpm
pnpm dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build

# pnpm
pnpm build
```
